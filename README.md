# Правила игры «Виселица»

## Настройка игры
1. **Выбор категории**: Пользователь выбирает категорию слов.
2. **Выбор сложности**: Пользователь выбирает уровень сложности (влияет на длину слов).
3. **Генерация слова**: Программа считывает файл со словами и случайным образом выбирает одно слово нужной категории и сложности.

## Интерфейс
На экране постоянно отображаются:
- **Состояние слова**: Загаданное слово в виде подчеркиваний и открытых букв (например: `м _ _ к _`).
- **Виселица**: Схематичное изображение человечка (ASCII-графика), которое дополняется после каждой ошибки.
- **Отображение ошибок**: Текущее ошибочно введенные буквы.

## Ход игры
1. **Ввод**: Пользователь вводит одну букву с клавиатуры.
2. **Обработка**:
    - **Буква есть в слове**: Все вхождения буквы открываются в слове.
    - **Буквы нет в слове**: Рисуется новая часть виселицы, счетчик попыток уменьшается.
    - **Буква уже была**: Ход не засчитывается, ошибка не рисуется.
    - **Неверный ввод**: Если введен не символ или более одной буквы, система просит повторить ввод.

## Условия завершения
- **Победа**: Все буквы слова открыты. Игра показывает поздравление и статистику.
- **Поражение**: Количество ошибок достигло максимума (виселица построена полностью). Игра показывает правильное слово и сообщение о проигрыше.
